# MilkData
Milk_database - база данных для задачи классификации цельного молока и молока с разным процентным содержанием примесей.
Описание данных находится в файле Данные.docx

crossval.py - скрипт для разбиения случайным образом 
базы данных на обучающую и тестовую выборки в заданном процентном соотношении.
How to use: crossval.py каталог_базаДанных 
В результате работы скрипта создается каталог database, который, в свою очередь, содержит
3 каталога: enroll, classes, 4recognition. По умолчанию для формирования пространства главных компонент используется 30% данных,
относящихся к разным классам; для формирования классов в признаковом пространстве - 50% данных. Остальное используется для валидации.
Каждый запуск скрипта формирует новый набор данных.

PCA_classification.exe - установщик программы для анализа данных методом kNN в пространстве главных компонент. 
How to use: proba_vine.exe полный_путь_до_enroll полный_путь_до_classes полный_путь_до_4recognition.
Результаты классификации находятся в файле result.txt в каталоге database.
